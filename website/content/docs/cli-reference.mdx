O Arc DevKit expõe um único entry point de CLI: **`arcdevkit`**.

## Setup

```bash
arcdevkit init                    # wizard .env interativo
arcdevkit status                  # conexão + info do bloco atual
arcdevkit config list             # todas as variáveis de ambiente
arcdevkit config get CHAVE        # ler uma variável específica
arcdevkit config set CHAVE VALOR  # escrever uma variável
```

## Carteira

```bash
arcdevkit wallet create                      # gerar nova carteira
arcdevkit wallet balance --address 0x...    # checar saldo
```

## AI Copilot

```bash
arcdevkit ask "O que é o Malachite consensus?"
arcdevkit ask --stream "Escreva um contrato USDC vault"
arcdevkit ask --json "..."                   # saída em JSON
```

## Debug de transações

```bash
arcdevkit debug tx 0xHash
arcdevkit debug tx 0xHash --abi Contrato.json
arcdevkit debug-batch hashes.txt            # analisar múltiplas txs
arcdevkit history                            # sessões recentes
arcdevkit history --limit 50
```

## Portfolio

```bash
arcdevkit portfolio analyze 0xCarteira
arcdevkit portfolio report wallets.json
```

## Pagamentos

```bash
arcdevkit send --to 0x... --amount 5.0 --token usdc
arcdevkit send --to 0x... --amount 1.0 --token native
```

## Flags globais

Funcionam em qualquer comando:

```bash
arcdevkit --json <cmd>    # saída legível por máquina (JSON)
arcdevkit -v <cmd>        # verbose (nível DEBUG de logging)
```

## Ajuda

```bash
arcdevkit --help
arcdevkit ask --help
arcdevkit debug --help
arcdevkit wallet --help
```

## Saída JSON

Todos os comandos suportam `--json` para integração com scripts:

```bash
arcdevkit status --json
# {"connected": true, "chain_id": 5042002, "block": 4821903, ...}

arcdevkit wallet balance --address 0x... --json
# {"native": "1.5", "usdc": "10.000000"}
```
